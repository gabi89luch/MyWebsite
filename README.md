# MyWebsite

A personal website for **Valentin Gabriel Luchian** built with HTML, CSS, Bootstrap, and JavaScript.

## Overview

This project contains a small static website that includes:

- `index.html` — Home page with profile, hobbies, education, media section, and links to contact/calculator pages
- `about.html` — About page with personal profile, education, skills, hobbies, and experience placeholder
- `contact.html` — Contact page with a simple client-side validated contact form
- `calculator.html` — Simple JavaScript calculator for basic arithmetic operations
- `css/styles.css` — Custom styles used across the site
- `images/` — Image assets used throughout the website
- `media/` — Embedded media content for the media section

## Features

- Responsive navigation built with Bootstrap
- Personal profile and CV-style sections
- Hobby cards and media examples
- Client-side form validation for contact submissions
- Interactive calculator with addition, subtraction, multiplication, and division support
- Clean layout using Bootstrap utility classes and custom CSS

## Pages

### Home
- Hero introduction
- Academic status as a computing student
- Favorite module highlight
- Hobbies section with cards
- Education timeline table
- Media section with a video and web tech overview

### About
- Personal profile snapshot
- Skills and interests overview
- Education details
- Work experience placeholder for future updates

### Contact
- Name, email, and message fields
- Basic validation on submit
- Success alert after form submission

### Calculator
- Number inputs and operation selector
- Results shown instantly on the page
- Division by zero handling

## Technologies

- HTML5
- CSS3
- Bootstrap 5.3
- Vanilla JavaScript

## Usage

To view the website, open `index.html` in a web browser. The site is static and does not require a backend server.

## Notes

- The contact form uses JavaScript validation and currently displays a browser alert on successful submission.
- The calculator runs entirely in the browser without external dependencies beyond Bootstrap.
